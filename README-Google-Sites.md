# Sharing Guide – Google Sites

Google Sites works best as a **wrapper / launch page** for this app.

## Recommended pattern

1. Host the React app first (GitHub Pages or another approved static host)
2. Copy the public HTTPS app URL
3. In Google Sites, use **Embed**
4. Paste the hosted app URL
5. Choose the largest layout / whole-page style available
6. Publish the Google Site

## Important embed requirements

- The app URL must be **HTTPS**
- The hosting target must allow **iframe embedding**
- If the host blocks embedding, link out from Google Sites instead of embedding inline

## Suggested layout for a training portal page

- Hero section: title + purpose
- Embedded app block
- Short facilitator instructions
- Discussion norms / workshop objective
- Optional pilot feedback form
